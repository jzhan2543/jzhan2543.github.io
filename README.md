# jzhan2543.github.io

Personal site for Jefferson Zhan — AI Engineer based in NYC.

Built with Astro + Tailwind v4. Static output.

## Develop

```sh
nvm use            # Node 20
corepack enable pnpm
pnpm install
pnpm dev           # http://localhost:4321
```

## Build

```sh
pnpm build         # → dist/
pnpm preview
```
